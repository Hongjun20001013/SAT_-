\section*{Course Overview}
This SAT course is designed to systematically cover core concepts, problem-solving strategies, and test-taking skills through review, practice, quizzes, and mock exams. Each week includes lectures, exercises, assignments, and reviews to ensure mastery.

\section*{Week One (June 9 -- June 13)}

\begin{itemize}[leftmargin=*]
    \item \textbf{First Lesson: Icebreaker and SAT Introduction}
    \begin{itemize}
        \item Introduce SAT (structure, timing, months of test dates)
        \item Introduction to Desmos calculator
        \item Overview of upcoming course schedule
        \item Topics: 1.1 -- 1.3
        \item In-class practice after concept review
        \item \textbf{Homework:} Challenging Practice 1.1--1.3 (Record time taken)
    \end{itemize}
    \item \textbf{Second Lesson:}
    \begin{itemize}
        \item Homework review 1.1--1.3
        \item New Topics: 1.4 -- 1.5
        \item In-class practice
        \item \textbf{Homework:} Challenging Practice 1.4--1.5 (Record time taken)
    \end{itemize}
    \item \textbf{Third Lesson:}
    \begin{itemize}
        \item Homework review 1.4--1.5
        \item Review Topics 1.1--1.5 (highlight common mistakes)
        \item New Topics: 2.1.1 -- 2.1.2
        \item In-class training and discussion
        \item \textbf{Homework:} Challenging Practice 2.1
    \end{itemize}
    \item \textbf{Friday:} Mock Test (Bluebook Practice 3, paper-based)
\end{itemize}
